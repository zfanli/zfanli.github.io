title: 48. Rotate Image (Medium)

```python
class Solution:
    def rotate(self, m: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        sz = len(m)

        for i in range(sz // 2 + sz % 2):
            for j in range(sz // 2):
                m[i][j], m[j][~i], m[~i][~j], m[~j][i] = m[~j][i], m[i][j], m[j][~i], m[~i][~j]
```
