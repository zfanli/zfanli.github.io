```java
class Solution {
    public int minPartitions(String n) {
        int ans = 0;
        for (char c : n.toCharArray()) {
            if (c - '0' > ans) ans = c - '0';
            if (ans == 9) return ans;
        }
        return ans;
    }
}
```

```python
class Solution:
    def minPartitions(self, n: str) -> int:
        return max(int(x) for x in n)
```
